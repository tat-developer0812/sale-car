### Website Overview Description

| Component | Description |
|-----------|-------------|
| **Car Catalog** | Browse cars with advanced filtering (brand, price, category, fuel type) |
| **Car Detail** | Full specifications, gallery, pricing, features for each vehicle |
| **Compare Tool** | Side-by-side comparison of 2-3 vehicles |
| **Blog/News** | Reviews, comparisons, guides, news, promotions |
| **Lead Forms** | Quick contact, test drive, quote request forms |
| **Brand Pages** | Dedicated pages for each car manufacturer |

### Sales Process Description

| Stage | Actions | Tools Used |
|-------|---------|------------|
| **Awareness** | Customer discovers website via Google, social media, or direct visit | SEO, Google Ads, Social Media |
| **Interest** | Customer browses cars, views details, reads specifications | Car catalog, filters, search |
| **Decision** | Customer compares cars, reads reviews, checks pricing | Compare tool, blog reviews |
| **Action** | Customer submits lead form, calls, or requests test drive | Contact forms, phone, WhatsApp |
| **Follow-up** | Sales team contacts within 5 minutes | Telegram, Email, Phone |
| **Conversion** | Schedule test drive, negotiate, close sale | CRM in Strapi |

### Technology Stack Description

| Layer | Technology | Purpose | Cost |
|-------|------------|---------|------|

| **UI Components** | Shadcn/ui | Pre-built accessible components | Free |
| **Forms** | React Hook Form + Zod | Form handling & validation | Free |
| **Backend/CMS** | Strapi | Content management, REST API | Free (self-hosted) |

| **Email** | Resend | Transactional emails | Free: 3K/month |
| **Notifications** | Telegram Bot | Real-time lead alerts | Free |
